FancyButton

Description: FancyButton is an upgraded, highly customizable UI button designed for professional UI applications and in-engine editors. It supports advanced styling, hover effects, rounded corners, borders, and custom fonts out of the box.

Features

  • Hover color transitions
  • Rounded corners with adjustable radius
  • Fully customizable borders
  • Custom fonts, sizes, and weights
  • Optional parent container support

Example: Usage

from VertexEngine import VertexUI

button = VertexUI.FancyButton(
    text="Play",
    width=200,
    height=50,
    bg_color=(40, 40, 40),
    hover_color=(60, 60, 60),
    text_color=(255, 255, 255),
    border_radius=12,
    border_color=(255, 255, 255),
    border_width=2,
    font_name="Arial",
    font_size=18,
    font_weight="bold"
)

Parameters

  • text (str, optional): Text displayed on the button
  • width (int, optional): Button width
  • height (int, optional): Button height
  • bg_color (tuple, optional): Background color
  • hover_color (tuple, optional): Color when hovering over the button
  • text_color (tuple, optional): Text color
  • border_radius (int, optional): Corner roundness
  • border_color (tuple, optional): Border color
  • border_width (int, optional): Border thickness
  • font_name (str, optional): Font family name
  • font_size (int, optional): Font size
  • font_weight (str, optional): Font weight (e.g. normal, bold)
  • parent (UI element, optional): Parent container

Returns

A FancyButton instance capable of handling click, hover, and interaction events.

Notes

  • Ideal for menus, editors, and in-game UI panels
  • Designed to feel modern and responsive
  • Can be extended for animations or sound effects
