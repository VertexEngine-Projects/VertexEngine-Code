InputField

Description: InputField is an upgraded and highly customizable text input component designed for UI applications and game editors. It allows users to enter text, supports placeholder hints, and offers full control over fonts, colors, and sizing.

Features

  • Placeholder text support
  • Custom fonts, sizes, and colors
  • Configurable width and height
  • Editor-friendly design
  • Keyboard input handling

Example: Usage

from VertexEngine import VertexUI

input_field = VertexUI.InputField(
    placeholder="Enter your name...",
    width=300,
    height=40,
    text_color=(255, 255, 255),
    bg_color=(30, 30, 30),
    font_name="Arial",
    font_size=16
)

Parameters

  • placeholder (str, optional): Grey hint text shown when the field is empty
  • width (int, optional): Width of the input field
  • height (int, optional): Height of the input field
  • text_color (tuple, optional): Color of entered text
  • bg_color (tuple, optional): Background color
  • font_name (str, optional): Font family
  • font_size (int, optional): Font size
  • parent (UI element, optional): Parent container

Returns

An InputField instance that stores and provides access to the user-entered text.

Notes

  • Ideal for forms, settings menus, and debug tools
  • Can be extended with validation or input masks
  • Works seamlessly with other VertexUI components
