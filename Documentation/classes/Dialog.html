Dialog

Description: Base dialog class built on QDialog. Supports modal and non-modal dialogs.

Parameters

  • parent (QWidget): Parent window
  • title (str): Dialog title
  • size (tuple): Width and height
  • modal (bool): Whether the dialog is modal

Functions

  • open_dialog() – Opens the dialog
  • close_dialog() – Closes the dialog

Returns

A dialog window.

MessageDialog

Description: Displays a simple message dialog with an OK button.

Example: Usage

MessageDialog(parent, "Hello World").open_dialog()

Returns

A modal message dialog.

ConfirmDialog

Description: Displays a confirmation dialog with Yes / No options.

Example: Usage

result = ConfirmDialog.ask(parent, "Are you sure?")
if result:
    print("User confirmed")

Functions

  • ask(parent, message, title="Confirm")
    Opens the dialog and returns True if accepted.

Returns

Boolean result.

InputDialog

Description: Displays a dialog that asks the user for text input.

Example: Usage

value = InputDialog.get(parent, "Enter your name:")
print(value)

Functions

  • value() – Returns the entered text
  • get(parent, label, title="Input")
    Opens the dialog and returns text or None.

Returns

User-entered text or None.

Dialogs

Description: A static helper class that provides quick access to common dialogs.

Example: Usage

Dialogs.message(parent, "Hello!")
confirmed = Dialogs.confirm(parent, "Continue?")
text = Dialogs.input(parent, "Your name:")

Functions

  • message(parent, text) – Shows a message dialog
  • confirm(parent, text) – Returns True / False
  • input(parent, text) – Returns text or None

Returns

Dialog results.
