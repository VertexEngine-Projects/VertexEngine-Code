GraphicalMath

Description: The GraphicalMath module provides mathematical tools for 2D vector operations. It includes the Vector2 class for vector math and UtilityFunctions for helper operations such as clamping, interpolation, and rounding.

Vector2

Description: Represents a 2D vector using (x, y) coordinates. Supports arithmetic operations, normalization, distance calculations, and geometric transformations like rotation.

Usage

from VertexEngine.math import Vector2

v1 = Vector2(3, 4)
v2 = Vector2(1, 2)

# Basic math
v3 = v1 + v2
v4 = v1 * 2

# Distance & normalization
distance = v1.distance_to(v2)
unit = v1.normalize()

# Rotation
rotated = v1.rotate(90)

Parameters

  • x (float): X-coordinate
  • y (float): Y-coordinate

Operators

  • v1 + v2: Add vectors
  • v1 - v2: Subtract vectors
  • v * scalar: Scale vector
  • v / scalar: Divide vector

Methods

  • magnitude(): Returns vector length
  • magnitude_squared(): Faster length squared
  • normalize(): Returns unit vector
  • dot(other): Dot product
  • distance_to(other): Distance between vectors
  • angle_to(other): Angle (degrees)
  • rotate(angle): Rotates vector

Returns

A Vector2 object representing the result of the operation.

UtilityFunctions

Description: A collection of helper functions for working with vectors and scalar values. These include clamping, interpolation, and rounding operations.

Usage

from VertexEngine.math import Vector2, UtilityFunctions

v = Vector2(10, 20)
min_v = Vector2(0, 0)
max_v = Vector2(15, 15)

utils = UtilityFunctions()

# Clamp vector
clamped = utils.clamp(min_v, max_v)

# Interpolate
v2 = Vector2(20, 40)
lerped = utils.lerp(v2, 0.5)

# Clamp magnitude
limited = utils.clamp_magnitude(10)

Methods

  • clamp(min_vec, max_vec): Clamp vector between bounds
  • clamp_magnitude(max_length): Limit vector length
  • lerp(other, t): Linear interpolation
  • floor(): Floor components
  • ceil(): Ceil components
  • round(): Round components
  • abs(): Absolute values

Static Methods

  • clamp_value(value, min, max): Clamp a number

Returns

Returns a modified Vector2 or scalar value depending on the function.
