BrowserTools.tabs

Description: The BrowserTools.tabs module provides a fully featured tab system for the Vertex WebEngine. It supports tab creation, closing, reordering, keyboard shortcuts, and automatic title updates.

This module is designed for browser-like applications, launchers, dashboards, and web-powered tools.

BrowserTabs

Description: A tab manager built on QTabWidget that hosts multiple WebEngine instances. Each tab represents an independent web page with its own state.

Features

  • Closable tabs
  • Draggable / reorderable tabs
  • Keyboard shortcuts
  • Dynamic tab titles
  • Safe tab removal

Default Behavior

  • Creates a default home tab on startup
  • Prevents closing the last remaining tab
  • Logs tab switches for debugging

Example: Usage

from BrowserTools.tabs import BrowserTabs

tabs = BrowserTabs()
tabs.show()

Returns

A tab-based web browser widget.

new_tab(url="vertex://home")

Description: Creates a new tab containing a WebEngine instance and loads the specified URL.

Parameters

  • url (str): URL to load. Supports vertex:// internal pages.

Behavior

  • Automatically switches to the new tab
  • Loads internal pages when using vertex://
  • Updates the tab title from the page title

Returns

None

close_tab(index)

Description: Closes a tab by index. The last remaining tab cannot be closed.

Parameters

  • index (int): Index of the tab to close

Safety

  • Prevents closing the final tab
  • Automatically deletes tab widgets

Returns

None

close_current_tab()

Description: Closes the currently active tab.

Returns

None

safe_remove(index)

Description: Removes a tab safely by checking index bounds before removal.

Parameters

  • index (int): Tab index

Returns

None

update_tab_title(engine, title)

Description: Updates the tab title when the page title changes. Titles are automatically truncated for readability.

Parameters

  • engine (WebEngine): The tab’s web engine
  • title (str): Page title

Returns

None

on_tab_changed(index)

Description: Triggered when the user switches tabs. Logs the active tab’s URL for debugging and tracking purposes.

Parameters

  • index (int): New active tab index

Returns

None

Keyboard Shortcuts

  • Ctrl + T – Open a new tab
  • Ctrl + W – Close the current tab

Notes

  • Each tab runs its own WebEngine
  • Supports internal and external URLs
  • Ideal for browser-based tools and editors
