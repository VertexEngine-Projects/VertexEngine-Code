LegacyText

Description: LegacyText is a basic text-rendering component for UI applications and game editors. It is intended for displaying static or minimally interactive text such as labels, captions, or debug information.

This component is currently a lightweight wrapper around PyQt6’s text-related widgets and follows their default behavior.

Status

  • ⚠️ Currently unfinished
  • API and behavior may change
  • Recommended for simple or experimental use

Example: Usage

from VertexEngine import VertexUI

label = VertexUI.LegacyText("Hello, Vertex!")

Parameters

  • [UNFINISHED]
    All parameters are optional. Most text-related attributes are self-explanatory (text content, color, font, size, alignment). For advanced configuration, refer to the PyQt6 text widget documentation.

Returns

A LegacyText instance that displays text on the screen.

Notes

  • Best used for labels, HUD elements, and debug text
  • Not intended for user text input
  • For advanced styling and layout control, future text components may be added
