AutoFontLabel

Description: A QLabel subclass that automatically adjusts its font size to fit the text within the label's width. The text is centered by default and will dynamically resize whenever the label's size changes.

Usage

from PyQt6.QtWidgets import QApplication
from VertexEngine.VertexUI import AutoFontLabel

app = QApplication([])

label = AutoFontLabel("Resizable Text")
label.resize(200, 50)
label.show()

app.exec()

Parameters

  • text (str, optional): Initial text to display. Defaults to an empty string.
  • parent (QWidget, optional): Parent widget. Defaults to None.

Methods

  • adjustFont(): Adjusts the font size to fit the text inside the current width of the label.

Events

  • resizeEvent(event): Automatically calls adjustFont() whenever the label is resized.

Behavior

  • Text is centered horizontally and vertically.
  • Font size starts at the label's height and decreases until the text fits the width.
  • Minimum font size is 1 pt.
