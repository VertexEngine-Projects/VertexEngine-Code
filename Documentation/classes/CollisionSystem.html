Collision System

Description: The Vertex collision system provides flexible and extensible collision detection between different shapes such as circles, polygons, and rotated rectangles. It uses polymorphism to automatically resolve collision checks between types.

Supported colliders:

  • Circle
  • Polygon (Triangles & Quads)
  • RotatedCollider (Rotated Rectangles)

Collider

Description: Base class for all colliders. It automatically determines the correct collision method based on the type of the other collider.

Usage

if collider1.collides_with(collider2):
    print("Collision detected!")

Returns: Boolean (True if colliding)

Circle

Description: A circular collider defined by a center position and radius. Efficient for simple collision detection.

Usage

c1 = Circle(100, 100, 50)
c2 = Circle(130, 100, 30)

if c1.collides_with(c2):
    print("Circles overlap!")

Parameters

  • x (float): X position
  • y (float): Y position
  • radius (float): Radius of the circle

Returns: Collision result (bool)

Polygon

Description: A polygon collider that supports triangles and quadrilaterals. Uses the Separating Axis Theorem (SAT) for accurate collision detection.

Usage

triangle = Polygon([(0,0), (100,0), (50,100)])
quad = Polygon([(200,200), (300,200), (300,300), (200,300)])

if triangle.collides_with(quad):
    print("Polygons collide!")

Parameters

  • points (list of tuples): 3–4 points defining the shape

Notes

  • Only triangles and quads are supported
  • Uses SAT for accurate edge-based collision
  • Supports collision with Circle and other Polygons

Returns: Collision result (bool)

RotatedCollider

Description: A rotated rectangular collider designed for accurate hit detection. Internally converted into a polygon for collision calculations.

Usage

rect1 = RotatedCollider(200, 200, 100, 50, angle=45)
rect2 = RotatedCollider(250, 200, 100, 50)

if rect1.collides_with(rect2):
    print("Rotated rectangles collide!")

Parameters

  • x (float): Center X position
  • y (float): Center Y position
  • width (float): Width of rectangle
  • height (float): Height of rectangle
  • angle (float): Rotation in degrees

Notes

  • Uses polygon conversion internally
  • More accurate than axis-aligned rectangles
  • Slightly more expensive computationally

Returns: Collision result (bool)

Example: Mixed Collision

circle = Circle(100, 100, 40)
rect = RotatedCollider(120, 100, 80, 60, angle=30)

if circle.collides_with(rect):
    print("Circle hit rotated rectangle!")

The system automatically determines the correct collision algorithm using polymorphism — no manual type checking required.
