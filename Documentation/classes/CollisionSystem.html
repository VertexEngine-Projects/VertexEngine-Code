Collider.collides_with(other)

Description: Checks whether this axis-aligned rectangle collides with another collider using the AABB (Axis-Aligned Bounding Box) method. Returns True if they overlap, False otherwise.

Usage

rect1 = Collider(10, 20, 50, 50)
rect2 = Collider(30, 40, 50, 50)
collision = rect1.collides_with(rect2)

Parameters

  • other (Collider): Another collider object to test against.

Returns

bool — True if colliding, False otherwise.

RotatedCollider.get_corners()

Description: Returns the four corner coordinates of this rectangle after applying rotation around its center. Useful for collision detection and rendering.

Usage

rect = RotatedCollider(100, 50, 40, 20, angle=30)
corners = rect.get_corners()

Parameters

None — uses the rectangle's internal properties.

Returns

list of tuples — A list of four (x, y) tuples representing each corner.

RotatedCollider.collides_with(other)

Description: Checks collision with another collider (rotated rectangle or circle) using SAT (Separating Axis Theorem). Returns True if they overlap.

Usage

rect1 = RotatedCollider(100, 50, 40, 20, angle=30)
rect2 = RotatedCollider(110, 60, 30, 10, angle=15)
collision = rect1.collides_with(rect2)

Parameters

  • other (Collider): Another collider to test collision against.

Returns

bool — True if colliding, False otherwise.

Circle.collides_with(other)

Description: Checks collision with another circle or polygon. Uses distance checking for circles and SAT for polygons.

Usage

c1 = Circle(50, 50, 10)
c2 = Circle(55, 55, 15)
collision = c1.collides_with(c2)

Parameters

  • other (Collider): Another collider object to check against.

Returns

bool — True if colliding, False otherwise.

Polygon.collides_with(other)

Description: Checks collision with another polygon or circle using SAT (Separating Axis Theorem) or edge projection. Supports triangles and quadrilaterals.

Usage

triangle = Polygon([(0,0),(50,0),(25,30)])
rect_poly = Polygon([(10,10),(60,10),(60,40),(10,40)])
collision = triangle.collides_with(rect_poly)

Parameters

  • other (Collider): Another collider object.

Returns

bool — True if colliding, False otherwise.
