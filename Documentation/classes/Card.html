Card

Description: A Card widget for UI layouts. It provides a styled container with rounded borders, padding, and a light background color. You can add other widgets inside the Card using addWidget().

Usage

from PyQt6.QtWidgets import QApplication
from VertexEngine.VertexUI import Card, VBox, FancyButton, Text

app = QApplication([])

card = Card()
card.addWidget(Text("Hello inside a card!"))
card.addWidget(FancyButton("Click Me"))

card.show()
app.exec()

Parameters

  • parent (QWidget, optional): The parent widget of the Card. Defaults to None.

Methods

  • addWidget(widget): Adds a child widget to the card's internal vertical layout (VBox).

Returns

The Card itself is a QWidget, so it can be placed in any layout or shown directly.

Style

  • Border: 1px solid #ccc
  • Border radius: 10px
  • Padding: 10px
  • Background color: #f9f9f9
