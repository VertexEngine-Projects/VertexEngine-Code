Rect

Description: Rect is a rectangle utility object used for positioning, sizing, and collision-style calculations within VertexEngine. Its parameters and behavior are fully compatible with pygame.Rect, allowing you to directly reuse or copy existing pygame code without modification.

Features

  • Identical parameter order to pygame.Rect
  • Useful for UI layout, positioning, and bounds
  • Lightweight and easy to construct
  • Beginner-friendly and familiar

Example: Usage

from VertexEngine import Rect

rect = Rect(50, 50, 200, 100)

Parameters

  • left (int, optional): X-coordinate of the rectangle
  • top (int, optional): Y-coordinate of the rectangle
  • width (int, optional): Rectangle width
  • height (int, optional): Rectangle height

Returns

A Rect object representing a rectangular area and its configuration.

Notes

  • Commonly used for layout calculations and UI positioning
  • Can be passed directly to drawing and collision-related functions
  • If you know pygame, you already know how to use this
